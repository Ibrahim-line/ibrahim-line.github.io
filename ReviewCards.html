Interactive Flashcards Review

What is HTML?
HyperText Markup Language
What does CSS do?
Styles the structure and layout of web pages.
What is a pseudo-class?
A keyword added to a selector that specifies a special state of the element.
What tag creates a link?
<a> tag (anchor)
What is responsive design?
Design that adjusts to different screen sizes.
What is a transition?
A way to animate changes to CSS properties smoothly.
What is an animation?
A way to change CSS properties in steps or over time.
What does the <head> tag contain?
Meta information about the HTML document.
